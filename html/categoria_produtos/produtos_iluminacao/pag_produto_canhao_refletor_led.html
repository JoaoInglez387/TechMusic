Descrição

Refletor Canhão 36 Leds Rgb

Este canhão de leds é ideal para festas ou eventos, proporcionando um espetáculo de iluminação com luzes de cores RGB. Com a função strobo e efeitos de luz ativados pelo sensor de batida musical, este equipamento eleva o nível de qualquer celebração.

Possui 7 canais DMX com diferentes funções de velocidade e intensidade de iluminação.

Funções do display de LED:

C001: Modo de controle 7CH (001-512)

CP01: Mudança de pulso colorido, transição gradual (01-50) B C botão de controle de velocidade

CC01: Mudança de cor rápida (B C botão de controle de velocidade)

CJ01: Transição de cores em grupo (01-50) (B C botão de controle de velocidade)

DENO: Transição de cores ativada por som
BEBE: Controle ativado por som

RGB: Ajuste de cor RBG (000-255) B C botão de controle de velocidade

ST: Strobo (B C botão de controle de velocidade)

Canais DMX - 7 Canais:

CH1: Intensidade de luz

CH2: Strobo

CH3: Seleção de cor

CH4: Ajuste de velocidade

CH5: Vermelho - Intensidade variável

CH6: Verde - Intensidade variável

CH7: Azul - Intensidade variável

Inclui duas alças para fixação superior.

Especificação:

- Strobo: 36 Leds - 12 vermelhos, 12 verdes, 12 azuis - Fonte de Luz: 36 Leds Rgb

- Consumo: 36W

- Sistema: RGB

- Conexões: Entrada e saída DMX

- Display Digital

- Alimentação: 110/220V

- Modos: Automático, rítmico, DMX512, sensor de som e Master-slave

- Dimensões aprox.: 18,5x10x17,5cm

- Peso aprox.: 600g

- Material: Plástico resistente
- Cor: Preto
